# Odin Recipes

A basic recipe website built as part of The Odin Project Foundations curriculum.

## About

This project demonstrates fundamental HTML concepts, including:

* HTML boilerplate structure
* Headings and paragraphs
* Lists (`ul` and `ol`)
* Images
* Links and navigation
* Relative file paths

The website contains several recipe pages, each with a description, ingredients list, preparation steps, and images.

## Recipes Included

* Asado
* Boeuf Bourguignon
* Carbonara
* Falafel
* Hummus
* Neapolitan Pizza
* Paella
* Sushi
* Tacos al Pastor

## Technologies Used

* HTML5
